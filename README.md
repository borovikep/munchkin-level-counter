# munchkin-level-counter
Счётчик уровней для игры в настольную игру Манчкин на React Native (Expo).
## Функции
- Добавление и редактирование игроков (имя, пол, цвет)
- Автоматический подсчёт силы (уровень + шмотки)
- Бросок кубика d6
- Тёмная и светлая тема
